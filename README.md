# SLEACT

## Description

slack과 유사한 서비스입니다.

NestJS와 TypeORM을 익히기 위한 프로젝트입니다.

<span style="color: #808080">front 부분은 제가 작성하지 않았습니다.</span>

**기능 소개**

-   로그인
-   워크스페이스 및 채널 생성
-   사용자 초대
-   채널 채팅방
-   다이렉트 메시지
-   이미지 업로드
